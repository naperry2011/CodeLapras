## Architecture Overview

*This section will describe the overall system architecture and design patterns.*

### Design Principles

- **Modular Architecture**: Each feature is self-contained
- **Event-Driven**: Components communicate via EventBus
- **Separation of Concerns**: Clear boundaries between data, logic, and UI
- **Vanilla JavaScript**: No framework dependencies (except xlsx for Excel)
